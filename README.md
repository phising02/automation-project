# Telegram Stock News Bot

This bot provides users with stock market news updates directly through Telegram. The bot fetches the latest news headlines and delivers them in a user-friendly format.

## Features
- **Real-time News Updates:** Get the latest stock market news as it happens.
- **User Notifications:** Customizable settings for how often to receive updates.
- **Multiple Stock Sources:** Integrates with various financial news APIs to provide comprehensive coverage.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/phising02/automation-project.git
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your Telegram bot token.

## Usage
- Run the bot by executing:
  ```bash
  python bot.py
  ```
